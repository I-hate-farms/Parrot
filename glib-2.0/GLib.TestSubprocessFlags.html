TestSubprocessFlags

Description:

[ Flags ]
public enum TestSubprocessFlags

Flags to pass to trap_subprocess to control input and output.

Note that in contrast with trap_fork, the default is to not show stdout and stderr.


Namespace: GLib
Package: glib-2.0

Content:

Enum values:

  • STDIN - If this flag is given, the child process will inherit the parent's stdin.
  • STDOUT - If this flag is given, the child process will inherit the parent's stdout.
  • STDERR - If this flag is given, the child process will inherit the parent's stderr.
