CollectType

Description:

public enum CollectType

A mixed enumerated type and flags field.

You must specify one type (string, strdup, boolean, tristate). Additionally, you may optionally bitwise OR the type with the flag g_markup_collect_optional.

It is likely that this enum will be extended in the future to support other types.


Namespace: GLib.Markup
Package: glib-2.0

Content:

Enum values:

  • INVALID - used to terminate the list of attributes to collect
  • STRING - collect the string pointer directly from the attribute_values[] array.
  • STRDUP - as with g_markup_collect_string, but expects a parameter of type (char **) and BusNames the returned pointer.
  • BOOLEAN - expects a parameter of type (gboolean *) and parses the attribute value as a boolean.
  • TRISTATE - as with g_markup_collect_boolean, but in the case of a missing attribute a value is set that compares equal to neither false nor true G_MARKUP_COLLECT_OPTIONAL is implied
  • OPTIONAL - can be bitwise ORed with the other fields.
