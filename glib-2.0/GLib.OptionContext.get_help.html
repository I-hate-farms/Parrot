get_help

Description:

public string get_help (bool main_help, OptionGroup? group)

Returns a formatted, translated help text for the given context.

To obtain the text produced by `--help`, call `g_option_context_get_help (context, TRUE, NULL)`. To obtain the text produced by `--help-all`, call `g_option_context_get_help (context, FALSE, NULL)`. To obtain the help text for an option group, call `g_option_context_get_help (context, FALSE, group)`.

Parameters:

this

a OptionContext
main_help

if true, only include the main group
group

the OptionGroup to create help for, or null

Returns:

A newly allocated string containing the help text
