LogLevelFlags

Description:

[ Flags ]
public enum LogLevelFlags

Flags specifying the level of log messages.

It is possible to change how GLib treats messages of the various levels using set_handler and set_fatal_mask.


Namespace: GLib
Package: glib-2.0

Content:

Enum values:

  • FLAG_RECURSION - internal flag
  • FLAG_FATAL - internal flag
  • LEVEL_ERROR - log level for errors, see error.
  • LEVEL_CRITICAL - log level for critical messages, see critical.
  • LEVEL_WARNING - log level for warnings, see warning
  • LEVEL_MESSAGE - log level for messages, see message
  • LEVEL_INFO - log level for informational messages, see info
  • LEVEL_DEBUG - log level for debug messages, see debug
  • LEVEL_MASK - a mask including all log levels
