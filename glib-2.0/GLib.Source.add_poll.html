add_poll

Description:

public void add_poll (ref PollFD fd)

Adds a file descriptor to the set of file descriptors polled for this source.

This is usually combined with Source to add an event source. The event source's check function will typically test the revents field in the PollFD struct and return true if events need to be processed.

This API is only intended to be used by implementations of SocketSource. Do not call this API on a SocketSource that you did not create.

Using this API forces the linear scanning of event sources on each main loop iteration. Newly-written event sources should try to use add_unix_fd instead of this API.

Parameters:

this

a SocketSource
fd

a PollFD structure holding information about a file descriptor to watch.
