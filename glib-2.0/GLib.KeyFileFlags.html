KeyFileFlags

Description:

[ Flags ]
public enum KeyFileFlags

Flags which influence the parsing.


Namespace: GLib
Package: glib-2.0

Content:

Enum values:

  • NONE - No flags, default behaviour
  • KEEP_COMMENTS - Use this flag if you plan to write the (possibly modified) contents of the key file back to a file; otherwise all comments will be lost when the key file is written back.
  • KEEP_TRANSLATIONS - Use this flag if you plan to write the (possibly modified) contents of the key file back to a file; otherwise only the translations for the current language will be written back.
