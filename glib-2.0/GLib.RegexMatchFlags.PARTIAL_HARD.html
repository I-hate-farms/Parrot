PARTIAL_HARD

Description:

PARTIAL_HARD

Turns on the partial matching feature.

In contrast to to PARTIAL_SOFT, this stops matching as soon as a partial match is found, without continuing to search for a possible complete match. See is_partial_match for more information. Since: 2.34
