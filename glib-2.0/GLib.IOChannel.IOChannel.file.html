IOChannel.file

Description:

public IOChannel.file (string filename, string mode) throws FileError

Open a file filename as a IOChannel using mode mode.

This channel will be closed when the last reference to it is dropped, so there is no need to call g_io_channel_close ( though doing so will not cause problems, as long as no attempt is made to access the channel after it is closed).

Parameters:

filename

A string containing the name of a file
mode

One of "r", "w", "a", "r+", "w+", "a+". These have the same meaning as in fopen

Returns:

A IOChannel on success, null on failure.
