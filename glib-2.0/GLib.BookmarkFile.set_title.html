set_title

Description:

public void set_title (string uri, string title)

Sets title as the title of the bookmark for uri inside the bookmark file this .

If uri is null, the title of this is set.

If a bookmark for uri cannot be found then it is created.

Parameters:

this

a BookmarkFile
uri

a valid URI or null
title

a UTF-8 encoded string
