VariantBuilder

Description:

public VariantBuilder (VariantType type)

Allocates and initialises a new VariantBuilder.

You should call g_variant_builder_unref on the return value when it is no longer needed. The memory will not be automatically freed by any other call.

In most cases it is easier to place a VariantBuilder directly on the stack of the calling function and initialise it with g_variant_builder_init.

Parameters:

type

a container type

Returns:

a VariantBuilder
