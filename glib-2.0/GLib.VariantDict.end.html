Description:

public Variant end ()

Returns the current value of this as a Variant of type g_variant_type_vardict, clearing it in the process.

It is not permissible to use this in any way after this call except for reference counting operations ( in the case of a heap-allocated VariantDict) or by reinitialising it with g_variant_dict_init (in the case of stack-allocated).
