BookmarkFileError

Description:

public errordomain BookmarkFileError

Error codes returned by bookmark file parsing.


Namespace: GLib
Package: glib-2.0

Content:

Error codes:

  • INVALID_URI - URI was ill-formed
  • INVALID_VALUE - a requested field was not found
  • APP_NOT_REGISTERED - a requested application did not register a bookmark
  • URI_NOT_FOUND - a requested URI was not found
  • READ - document was ill formed
  • UNKNOWN_ENCODING - the text being parsed was in an unknown encoding
  • WRITE - an error occurred while writing
  • FILE_NOT_FOUND - requested file was not found
