Description:

public errordomain IOChannelError

Error codes returned by IOChannel operations.


Namespace: GLib
Package: glib-2.0

Content:

Error codes:

  • FBIG - File too large.
  • INVAL - Invalid argument.
  • IO - IO error.
  • ISDIR - File is a directory.
  • NOSPC - No space left on device.
  • NXIO - No such device or address.
  • OVERFLOW - Value too large for defined datatype.
  • PIPE - Broken pipe.
  • FAILED - Some other error.
