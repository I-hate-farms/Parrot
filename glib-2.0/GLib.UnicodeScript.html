UnicodeScript

Description:

public enum UnicodeScript

The UnicodeScript enumeration identifies different writing systems.

The values correspond to the names as defined in the Unicode standard. The enumeration has been added in GLib 2.14, and is interchangeable with PangoScript.

Note that new types may be added in the future. Applications should be ready to handle unknown values. See <ulink url="http://www.unicode.org/reports/tr24/">Unicode Standard Annex #24: Script names</ulink>.


Namespace: GLib
Package: glib-2.0
