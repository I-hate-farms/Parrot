FormatSizeFlags

Description:

[ Flags ]
public enum FormatSizeFlags

Flags to modify the format of the string returned by format_size.


Namespace: GLib
Package: glib-2.0

Content:

Enum values:

  • DEFAULT - behave the same as g_format_size
  • LONG_FORMAT - include the exact number of bytes as part of the returned string.
  • IEC_UNITS - use IEC (base 1024) units with "KiB"-style suffixes.
