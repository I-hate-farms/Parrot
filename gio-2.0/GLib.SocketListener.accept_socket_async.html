accept_socket_async

Description:

public async Socket accept_socket_async (Cancellable? cancellable = null, out Object source_object = null) throws Error

This is the asynchronous version of accept_socket.

When the operation is finished callback will be called. You can then call accept_socket_async.end to get the result of the operation.
