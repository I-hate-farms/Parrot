Cancellable

Object Hierarchy:

Description:

public class Cancellable : Object

GCancellable is a thread-safe operation cancellation stack used throughout GIO to allow for cancellation of synchronous and asynchronous operations.


Namespace: GLib
Package: gio-2.0
