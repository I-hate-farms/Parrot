DBusObjectSkeleton

Object Hierarchy:

Description:

public class DBusObjectSkeleton : Object, DBusObject

A DBusObjectSkeleton instance is essentially a group of D-Bus interfaces.

The set of exported interfaces on the object may be dynamic and change at runtime.

This type is intended to be used with DBusObjectManager.
