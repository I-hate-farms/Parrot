SimpleAction.stateful

Description:

public SimpleAction.stateful (string name, VariantType? parameter_type, Variant state)

Creates a new stateful action.

state is the initial state of the action. All future state values must have the same VariantType as the initial state.

If the state GVariant is floating, it is consumed.

Parameters:

name

the name of the action
parameter_type

the type of the parameter to the activate function
state

the initial state of the action

Returns:

a new SimpleAction
