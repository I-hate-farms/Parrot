NetworkMonitor

Object Hierarchy:

Description:

public interface NetworkMonitor : Object, Initable

NetworkMonitor provides an easy-to-use cross-platform API for monitoring network connectivity.

On Linux, the implementation is based on the kernel's netlink interface.
