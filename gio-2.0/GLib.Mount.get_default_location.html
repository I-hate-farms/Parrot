get_default_location

Description:

public abstract File get_default_location ()

Gets the default location of this.

The default location of the given this is a path that reflects the main entry point for the user (e.g. the home directory, or the root of the volume).

Parameters:

this

a Mount.

Returns:

a File. The returned object should be unreffed with unref when no longer needed.
