DBUS_ERROR

Description:

DBUS_ERROR

A remote object generated an error that doesn't correspond to a locally registered Error error domain.

Use get_remote_error to extract the D-Bus error name and strip_remote_error to fix up the message so it matches what was received on the wire. Since 2.26.
