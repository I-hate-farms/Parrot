PasswordSave

Description:

public enum PasswordSave

PasswordSave is used to indicate the lifespan of a saved password.

Gvfss stores passwords in the Gnome keyring when this flag allows it to, and later retrieves it again from there.


Namespace: GLib
Package: gio-2.0

Content:

Enum values:

  • NEVER - never save a password.
  • FOR_SESSION - save a password for the session.
  • PERMANENTLY - save a password permanently.
