InetSocketAddress.from_string

Description:

public InetSocketAddress.from_string (string address, uint port)

Creates a new InetSocketAddress for address and port.

If address is an IPv6 address, it can also contain a scope ID (separated from the address by a "<literal>%</literal >").

Parameters:

address

the string form of an IP address
port

a port number

Returns:

a new InetSocketAddress, or null if address cannot be parsed.
