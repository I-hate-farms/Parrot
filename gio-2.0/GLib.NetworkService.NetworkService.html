Description:

public NetworkService (string service, string protocol, string domain)

Creates a new NetworkService representing the given service, protocol, and domain.

This will initially be unresolved; use the SocketConnectable interface to resolve it.

Parameters:

service

the service type to look up (eg, "ldap")
protocol

the networking protocol to use for service (eg, "tcp")
domain

the DNS domain to look up the service in

Returns:

a new NetworkService
