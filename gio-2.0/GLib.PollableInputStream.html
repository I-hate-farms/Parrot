PollableInputStream

Object Hierarchy:

Description:

public interface PollableInputStream : InputStream

PollableInputStream is implemented by InputStream s that can be polled for readiness to read.

This can be used when interfacing with a non-GIO API that expects UNIX-file-descriptor-style asynchronous I/O rather than GIO-style.
