GET_INVALIDATED_PROPERTIES

Description:

GET_INVALIDATED_PROPERTIES

If set, the property value for any <emphasis>invalidated property</emphasis> will be (asynchronously) retrieved upon receiving the <ulink url="http://dbus.

freedesktop.org/doc/dbus-specification.html#standard-interfaces-properties">PropertiesChanged</ulink> D-Bus signal and the property will not cause emission of the g_properties_changed signal. When the value is received the g_properties_changed signal is emitted for the property along with the retrieved value. Since 2.32.
