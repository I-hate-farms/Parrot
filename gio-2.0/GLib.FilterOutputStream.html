FilterOutputStream

Object Hierarchy:

Description:

public abstract class FilterOutputStream : OutputStream

Base class for output stream implementations that perform some kind of filtering operation on a base stream.

Typical examples of filtering operations are character set conversion, compression and byte order flipping.
