get_display

Description:

public virtual string get_display (AppInfo info, List<File> files)

Gets the display string for the this.

This is used to ensure new applications are started on the same display as the launching application, by setting the `DISPLAY` environment variable.

Parameters:

this

a AppLaunchContext
info

a AppInfo
files

a List of File objects

Returns:

a display string for the display.
