proxy_enumerate

Description:

public abstract SocketAddressEnumerator proxy_enumerate ()

Creates a SocketAddressEnumerator for this that will return GProxyAddressess for addresses that you must connect to via a proxy.

If this does not implement proxy_enumerate, this will fall back to calling enumerate.

a SocketConnectable
