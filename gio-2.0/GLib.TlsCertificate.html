TlsCertificate

Object Hierarchy:

Description:

public class TlsCertificate : Object

A certificate used for TLS authentication and encryption.

This can represent either a certificate only (eg, the certificate received by a client from a server), or the combination of a certificate and a private key (which is needed when acting as a TlsServerConnection).
