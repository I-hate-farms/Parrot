query_info

Description:

public virtual FileInfo query_info (string attributes, Cancellable? cancellable = null) throws Error

Queries a file input stream the given attributes.

This function blocks while querying the stream. For the asynchronous (non-blocking) version of this function, see query_info_async. While the stream is blocked, the stream will set the pending flag internally, and any other operations on the stream will fail with g_io_error_pending.

Parameters:

this

a FileInputStream.
attributes

a file attribute query string.
cancellable

optional Cancellable object, null to ignore.

Returns:

a FileInfo, or null on error.
