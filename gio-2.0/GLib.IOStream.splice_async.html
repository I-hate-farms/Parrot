splice_async

Description:

public async bool splice_async (IOStream stream2, IOStreamSpliceFlags flags, int io_priority = DEFAULT, Cancellable? cancellable = null) throws Error

Asyncronously splice the output stream of this to the input stream of stream2, and splice the output stream of stream2 to the input stream of this.

When the operation is finished callback will be called. You can then call splice_async.end to get the result of the operation.

a IOStream.
stream2
