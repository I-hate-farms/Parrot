SocketAddress

Object Hierarchy:

Description:

public abstract class SocketAddress : Object, SocketConnectable

SocketAddress is the equivalent of struct sockaddr in the BSD sockets API.

This is an abstract class; use InetSocketAddress for internet sockets, or GUnixSocketAddresss for UNIX domain sockets.
