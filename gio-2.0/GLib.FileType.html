Description:

public enum FileType

Indicates the file's on-disk type.


Namespace: GLib
Package: gio-2.0

Content:

Enum values:

  • UNKNOWN - File's type is unknown.
  • REGULAR - File handle represents a regular file.
  • DIRECTORY - File handle represents a directory.
  • SYMBOLIC_LINK - File handle represents a symbolic link (Unix systems).
  • SPECIAL - File is a "special" file, such as a socket, fifo, block device, or character device.
  • SHORTCUT - File is a shortcut (Windows systems).
  • MOUNTABLE - File is a mountable location.
