FilesystemPreviewType

Description:

public enum FilesystemPreviewType

Indicates a hint from the file system whether files should be previewed in a file manager.

Returned as the value of the key FILE_ATTRIBUTE_FILESYSTEM_USE_PREVIEW.


Namespace: GLib
Package: gio-2.0

Content:

Enum values:

  • IF_ALWAYS - Only preview files if user has explicitly requested it.
  • IF_LOCAL - Preview files if user has requested preview of "local" files.
  • NEVER - Never preview files.
