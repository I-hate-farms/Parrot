DBusObject

Object Hierarchy:

Description:

public interface DBusObject : Object

The DBusObject type is the base type for D-Bus objects on both the service side (see DBusObjectSkeleton) and the client side (see DBusObjectProxy).

It is essentially just a container of interfaces.
